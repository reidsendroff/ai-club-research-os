# Northern Highlands AI Club — Operating System

**Version 1.0 | Academic Year 2026–2027**
**Maintained by Reid Sendroff & Ben Shamosh**

---

## What This Is

This is the complete operating system for the Northern Highlands AI Club. Every process, every decision framework, every template, every protocol lives here. If you're running this club and you have a question about how something should work, the answer is in one of these files.

This is a living document. It gets updated when processes change, when something stops working, or when a new officer takes over and needs to understand how the club runs.

## How to Use This OS

**If you're a new officer:** Read files 01, 02, and your role's section in 02. Then read the runbook for the first thing you'll be responsible for.

**If you're running a meeting:** Open `03_Meeting_Runbook.md`. Everything you need is there.

**If you need to write an announcement:** Open `06_Announcement_Playbook.md`.

**If you're onboarding a new member:** Open `04_Member_System.md`.

**If you're graduating:** Read `10_Succession_Protocol.md` six weeks before your last meeting.

---

## The One Rule

If you do something that works and it isn't written here, write it here. The only way this document stays useful is if people add to it when they learn something.
